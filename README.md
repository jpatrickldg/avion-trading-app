# Stock Trading App

## Built With

- [Ruby on Rails](https://rubyonrails.org/)
- [PostgreSQL](https://www.postgresql.org/)
- [TailwindCSS](https://tailwindcss.com/)

## Version
```
* Ruby 3.1.2
* Rails 7.0.4
```

## Setup
```
 $ bundle install
 $ rails db:setup
```
 
## Starting the application
```
 $ rails server
```
